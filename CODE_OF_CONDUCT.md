# Code Of Conduct

This project expects respectful, professional collaboration.

## Expected Behavior

- Be constructive and specific.
- Discuss technical disagreements with evidence.
- Respect privacy and do not publish secrets, credentials, license keys, private server details, or exploit details.
- Keep issues and pull requests focused on the project.

## Unacceptable Behavior

- Harassment, abuse, or discriminatory language.
- Publishing private security details or personal information.
- Spam, trolling, or intentionally disruptive behavior.

Maintainers may edit, hide, or remove contributions that violate these expectations.
